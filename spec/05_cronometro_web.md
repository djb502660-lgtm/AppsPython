# Spec 05: Cronómetro Web

## Objetivo
Crear una página web con Flask que muestre un cronómetro (iniciar, pausar, reiniciar) usando JavaScript básico.

## Tecnologías
- **Frontend:** HTML, Bootstrap 5, JavaScript (`setInterval`).
- **Backend:** Flask para servir la plantilla.

## Alcance
- Botones: Iniciar, Pausar, Reiniciar.
- Mostrar tiempo en formato `00:00:00`.
- **Ubicación:** `app/cronometro_web/`.

## Pasos
1. `requirements.txt` con Flask.
2. `app.py` con una ruta `/`.
3. `templates/index.html` y `static/script.js`.
